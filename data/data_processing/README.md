# Парсер клинических рекомендаций RUSSCO

Скрипт для автоматической загрузки и разметки PDF-файлов с сайта Российского общества клинической онкологии.

## Как использовать
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите: `python russco_parser.py`
3. Результат появится в `russco_chunks.json`

## Что получаем на выходе
- Чанки (смысловые куски текста)
- Метаданные: тип рака, заголовок, источник
- JSON-формат, готовый для векторной базы
