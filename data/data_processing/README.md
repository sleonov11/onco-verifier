# Парсер клинических рекомендаций RUSSCO

Скрипт для загрузки и разметки PDF-файлов с клиническими рекомендациями 
Российского общества клинической онкологии (RUSSCO).

## ПОЛУЧЕНИЕ PDF-ФАЙЛОВ:

### Откуда брать файлы:
Сайт RUSSCO: https://rosoncoweb.ru/standarts/RUSSCO/?ysclid=mltovvj5oo394201940

### Как скачивать:
Сайт защищён от автоматических запросов (ошибка 403), поэтому **автоматическое скачивание НЕ работает**. Нужно скачать PDF **вручную**:

### Куда сохранять:
Все скачанные PDF должны лежать в папке **`russco_pdfs/`** в **корне проекта**.

**Правильный путь:** `onco-verifier/russco_pdfs/`

## ЧТО ПОЛУЧИТСЯ НА ВЫХОДЕ

### Где окажутся метаданные:
После запуска скрипта в **корне проекта** появится файл onco-verifier/russco_chunks.json

### Какие метаданные добавляются:
Для каждого кусочка текста (чанка) скрипт добавляет "Поле", "Что означает", "Откуда берётся"

**Запускать строго из корня проекта**
